# 🍔 Landing Page — King Burger

> 🎓 Projeto desenvolvido durante a **Imersão Dev do Futuro** da [Dev em Dobro](https://devemdobro.com)

Este projeto consiste no desenvolvimento de uma landing page para uma rede de hamburguerias, criada como prática dos meus estudos em HTML, CSS e JavaScript.

A proposta foi aplicar conceitos de estrutura semântica, organização de layout, responsividade e interatividade, criando uma interface moderna e apetitosa para apresentar o cardápio, promoções e diferenciais do King Burger.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Estruturação semântica

---

## 🎨 Funcionalidades implementadas

- Menu de navegação superior fixo
- Hero section com destaque para promoção principal (KB Ninja)
- Seção de combos em destaque com carrossel de produtos
- Vitrine de ingredientes premium com ícones visuais
- Seção de produto especial com descrição (Crispy Chicken Nuggets)
- Banner promocional temático (O Poder Ninja chegou ao BK!)
- Menu completo com categorias filtráveis (Burgers, Bebidas, Acompanhamentos)
- Cards de produtos com preços e descrições
- Seção de download do aplicativo
- Layout responsivo para diferentes telas
- Efeitos hover e animações em elementos

---

## 🎯 Objetivo do projeto

Praticar o desenvolvimento de interfaces modernas para web, aplicando conceitos de:

- Estrutura semântica em HTML
- Estilização com CSS
- Layout com Grid e Flexbox
- Responsividade com Media Queries
- Interatividade com JavaScript (filtros de menu, carrosséis)

---
